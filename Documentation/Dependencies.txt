|Production dependencies

|-cookie-parser: allows easy cookie handling.
|-dotenv: to handle environment variables.
|-express: backend nodejs server framework.
|-express-session: to handle session info using cookies.
|-express-handlebars: allows handlebars view engine rendering from server side.
|-mongoose- MongoDB ODM framework.

|Dev dependencies
|-mocha: to run unit testing.
|-nodemon: to restart server every time a change is made.

npm i cookie-parser dotenv express express-session express-handlebars mongoose & npm i mocha nodemon -D
