Everything code related is contained inside src folder.
./src/<folder>:
  config: contains general configuration of the software instance.
  controller: entity database controller .js files.
  libs: general handlers and init code scripts.
  model: entity database model .js files.
  public: public accessible files.
  routes: file scripts that contain web route declarations and logic.
  test: Mocha testing file scripts.
  views: view engine and html client side viewable files.

./src/<script>.js
  index.js: initiates server and other web entities such as the database.
  server.js: contains all server config.
